## Overview

Loopback Advanced is a virtual audio routing utility from Rogue Amoeba that creates virtual audio devices which combine and route audio from applications, physical inputs, and other sources into configurable audio streams. It solves the fundamental audio routing limitation on macOS — the inability to direct audio output from one application into another as input — enabling podcasting, streaming, recording, and audio production workflows that require flexible audio routing.

Virtual audio devices appear as standard audio devices that any application can select as input or output. The virtual devices combine audio from multiple sources — application output, microphone input, system sounds — into a single stream that recording, streaming, or conferencing applications can capture.

Visual routing with a cable-based interface connects audio sources to virtual device channels with volume controls, panning, and monitoring at each connection point. The visual approach makes complex routing configurations understandable at a glance and adjustable with drag-and-drop simplicity.

Per-application audio capture routes audio from specific applications without capturing all system audio, enabling selective recording of application output alongside microphone input for podcasts, tutorials, and screen recordings.

For podcasters, streamers, musicians, and anyone who needs to route audio between applications on macOS, Loopback Advanced is the audio routing tool that makes macOS audio as flexible as professional studio routing.

## Performance

Loopback balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
